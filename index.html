Heart Disease Prediction Portfolio

By Ines Marin

Project Summary

This project demonstrates the use of classification models, including logistic regression, decision trees, and ensemble methods, to predict the likelihood of heart disease.

Methodology

  • Data preprocessing and exploratory analysis
  • Model training and validation
  • Performance evaluation using AUC, accuracy, and precision

Conclusion

The best-performing model achieved high accuracy and provided interpretable insights into key health indicators linked to heart disease risk.
